# Linear Programming and Data Products

## Setup

To run the examples yourself use the following setup

```bash
# Change to the checked out directory
cd  ~/.../.../lp_for_data_products

# Create a virtual environment
python3 -m venv lp_dp_env
source lp_dp_env/bin/activate

# Install all dependencies
python -m pip install -r requirements.txt

# Start jupyter
jupyter lab
```

## Overview

*basic_examples.ipynb* contains a small problem to showcase how to work with the pulp library
